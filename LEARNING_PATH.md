# 🎓 DevOps Learning Path

**Repository:** DevOps-Projects  
**Goal:** Master DevOps skills through hands-on projects

## 📋 Learning Modules

### **Module 1: Linux Fundamentals** ⭐ Beginner
**Project:** Fun with Linux for Cloud & DevOps Engineers

**Topics Covered:**
- Linux command line basics
- User and group management
- File systems and permissions
- Process management
- Networking basics

**Prerequisites:** None
**Duration:** 1-2 hours

**Files:**
- Fun with Linux/001-Linux-Fundamentals.md
- Fun with Linux/002-User-Management.md
- Fun with Linux/003-File-Permissions.md
- Fun with Linux/004-Process-Management.md
- Fun with Linux/005-Package-Management.md
- Fun with Linux/006-Networking-Basics.md
- Fun with Linux/007-Practical-Exercises.md

### **Module 2: Docker Containerization** 🐳 Beginner-Intermediate
**Project:** DevOps-Project-01 - Django on AWS ECS/ECR

**Topics Covered:**
- Docker basics and Dockerfiles
- Container registries (ECR)
- AWS ECS deployment
- Container orchestration fundamentals

**Prerequisites:** Linux basics (Module 1)
**Duration:** 2-3 hours

**Files:**
- DevOps-Project-01/Docker-101.md
- DevOps-Project-01/Docker-102-Images.md
- DevOps-Project-01/Docker-103-Container-Lifecycle.md
- DevOps-Project-01/Docker-104-Dockerfile-Best-Practices.md
- DevOps-Project-01/Docker-105-AWS-ECR.md
- DevOps-Project-01/Docker-106-EC2-ECS.md
- DevOps-Project-01/Docker-107-Compose-Practice.md

---

### **Module 3: CI/CD Fundamentals** 🔄 Beginner
**Project:** DevOps-Project-02 - Jenkins on AWS

**Topics Covered:**
- CI/CD pipeline concepts
- Jenkins installation and configuration
- Maven builds and dependencies
- GitHub integration
- Docker deployments in pipelines

**Prerequisites:** Docker knowledge (Module 2)
**Duration:** 2-3 hours

---

### **Module 4: Infrastructure as Code** 🏗️ Intermediate
**Project:** DevOps Project-11 - Two-Tier AWS Infrastructure

**Topics Covered:**
- Terraform basics
- AWS VPC and networking
- Load Balancers (ALB)
- ECS services
- RDS databases
- Route 53 DNS
- WAF and CloudFront
- ACM certificates
- IAM roles and policies

**Prerequisites:** Basic AWS knowledge
**Duration:** 3-4 hours
