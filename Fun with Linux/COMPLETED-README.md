# Module 1: Linux Fundamentals - Learning Path

**Project:** Fun with Linux for Cloud & DevOps Engineers

---

## Files Created (in order)

1. **001-Linux-Fundamentals.md** - File system basics, viewing, copying, compressing
2. **002-User-Management.md** - Creating users/groups, managing accounts
3. **003-File-Permissions.md** - chmod, chown, special permissions
4. **004-Process-Management.md** - Process monitoring, services, troubleshooting
5. **005-Package-Management.md** - apt/dnf, package archives
6. **006-Networking-Basics.md** - Network config, diagnostics, SSH, firewall
7. **007-Practical-Exercises.md** - 7 hands-on exercises

---

## How to Follow This Module

### Option 1: Read in Order
Open each file from 001 to 007 and work through them sequentially.

### Option 2: Practice in Terminal
- Open a terminal
- Try each command in the files on your own system
- Complete the exercises

### Option 3: Study with Examples
Read through the files, then practice on:
- Your local machine
- A VM/VPS (e.g., Ubuntu, Debian)
- Cloud instance (AWS EC2, Azure VM)
